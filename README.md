# Web Crawler in Twitter
자바를 활용한 트위터용 웹 크롤러 만들기

### 목적
내가 조회하고자 하는 단어를 등록해두면 트위터에서 해당 단어가 포함된 트위터가 수집되도록 한다.

### 구조
스케줄러 --> Twitter real-time stream API 를 통해서 원하는 키워드 수집 --> 가공 --> json 형태로 엘라스틱서치에 저장
API에서 원하는 형태로만 조회 가능하도록
